## Responsive Breakpoints

### Mobile (320px - 768px)
```
- Full-width form
- Single column layout
- Larger touch targets (48px minimum)
- Larger text for readability
- Stacked sections
```

### Tablet (768px - 1024px)
```
- Two-column layout starts
- Optimized spacing
- Balanced proportions
```

### Desktop (1024px+)
```
- Two-column layout (form + image)
- Form: 50% width
- Image: 50% width
- Max width: 1120px centered
```

## Animation & Transitions

### Countdown Timer
```
Property: width, backgroundColor
Duration: 1000ms (smooth per-second update)
Effect: Progress bar fills left-to-right
```

### Button Hover
```
Property: backgroundColor
Duration: 200ms
Easing: ease-in-out
```

### Success Message
```
Property: opacity, translateY
Duration: 300ms
Effect: Fade in from top
```

### OTP Input Focus
```
Property: borderColor, boxShadow
Duration: 200ms
Effect: Blue ring highlight
```
